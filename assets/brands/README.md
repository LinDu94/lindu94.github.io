# 相机品牌 Logo 图片

请在此目录下放置相机品牌的 logo 图片文件。

## 支持的品牌

系统会自动识别以下品牌并显示对应的 logo：

- `nikon.svg` - Nikon 相机
- `canon.svg` - Canon 相机
- `sony.svg` - Sony 相机
- `fujifilm.svg` - Fujifilm 相机
- `hasselblad.svg` - Hasselblad 相机
- `leica.svg` - Leica 相机
- `panasonic.svg` - Panasonic 相机
- `olympus.svg` - Olympus 相机
- `pentax.svg` - Pentax 相机
- `dji.svg` - DJI 无人机

## 图片要求

- **格式**：SVG（矢量图，推荐）
- **尺寸**：SVG 会自动缩放，建议设计时使用 32x32 到 64x64 像素的视图框
- **背景**：透明背景
- **命名**：使用小写字母，如 `nikon.svg`
- **颜色**：建议使用单色或品牌标准色，确保在深色背景下清晰可见

## 获取 Logo

您可以从以下来源获取品牌 logo：

1. 品牌官方网站
2. 品牌资源中心
3. 使用在线 SVG 编辑器创建或优化
4. 从品牌官方媒体资源包下载 SVG 格式

## 注意事项

- 如果 logo 图片不存在，系统会自动隐藏 logo，只显示相机型号
- Logo 会自动适配大小（16x16px）
- 确保图片文件权限允许访问
